# radar-dados-cloud
- Pipeline automatizado que monitora fontes técnicas (Medium/RSS) - Processamento com Google Gemini (IA) para gerar resumos críticos - Sistema de memória (Firestore) para evitar duplicação - Entrega via Telegram API - Orquestração com Cloud Scheduler (roda sozinho todo dia às 8h) - Custo operacional: R$ 0,00 (Free Tier)
